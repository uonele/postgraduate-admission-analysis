# 招考数据深潜分析中心

欢迎使用大厂（国企）招考全链路数据分析与追踪知识库！

本站点主要用于沉淀豫信电科集团及下属子公司的社招/校招录取全链路数据洞察，通过穿透公告名单数据，分析各层人才被淘汰与录用真实分布情况。

## 导航指引

- 🏢 **[五大公司全链路漏斗分析](24年/五大公司全链路总览表.md)**：包含最新 2025/2026 年度名校和双非学生的录用转化率，直击招考核心现象（学历悬殊与倒挂）；
- 📈 **人才梯队结构化分析**：对研发类、投资类不同维度的群体进行剖析；
- 📂 **历史追溯数据**：基于去年数据的深入复盘；

**核心结论预览：**

> 在最新一轮大规模招考中发现严重的“学历倒挂”：淘汰百人大关中有极其庞大比例的 **985 (如南大、复旦、人大) 和 211名校 生存率惨不忍睹**，进入复试甚至最终录用的人员很多都是双非、普通二本出身。这展现了该公司面试体系极度倾向于“实用派”及往期经验。具体数据见导航。
